Vnet creation:
 az network vnet create -n gopalvnet2 -g gopal

subnet gateway creation:
az network vnet subnet create --vnet-name gopalvnet -n GatewaySubnet -g gopal --address-prefix 10.0.0.32/28

subnet backend and front creation:
az network vnet subnet create -g gopal --vnet-name gopalvnet -n backend --address-prefixes 10.0.0.16/28
az network vnet subnet create  -g gopal --vnet-name gopalvnet -n front --address-prefixes 10.0.0.0/24
